# Launcher icons

Place `ic_launcher.png` files here at the appropriate densities:
- mipmap-mdpi: 48x48
- mipmap-hdpi: 72x72
- mipmap-xhdpi: 96x96
- mipmap-xxhdpi: 144x144
- mipmap-xxxhdpi: 192x192

The Flutter project does not require icons to build, but a missing icon
will cause Android to fall back to the system default.

You can regenerate icons automatically using the `flutter_launcher_icons`
package — see `pubspec.yaml` for the optional dev_dependency.
