# Web assets

Place the following files here (regenerated by `flutter create .` if missing):

- `favicon.png` — browser tab icon
- `icons/Icon-192.png` — PWA icon 192x192
- `icons/Icon-512.png` — PWA icon 512x512
- `icons/Icon-maskable-192.png` — maskable PWA icon 192x192
- `icons/Icon-maskable-512.png` — maskable PWA icon 512x512

A simple way to generate all of these is the `flutter_launcher_icons` package
— add it as a dev_dependency and run `dart run flutter_launcher_icons`.
